# Privacy and Local-Only Data Model (Task 16)

StreakQuit stores user data locally on the device using AsyncStorage.

## What is Stored Locally
- Onboarding preferences and reasons.
- Craving logs, relapse logs, journal entries.
- Notification preferences.
- Premium status fields.

## What Is Not Collected by Default
- No account system.
- No cloud sync.
- No analytics SDK configured in app runtime.
- No ad tracking identifiers required for core usage.

## User Controls
- App reset in Settings clears local data and restarts onboarding.
- Users can disable notifications at OS level.

## Legal Copy Guidance
Use clear language in store listing and in-app legal section:
"Your quit data stays on your device by default."
